# AI Native Core --- Web

This is the Next.js frontend for the ai-native-core monorepo.

It will serve as:

- The primary web UI
- A streaming-first AI interface
- A debugging surface for AI runtime behavior
- A reusable frontend template for AI-native products

---

## Tech Stack

- Next.js
- React
- TypeScript
- Turborepo
